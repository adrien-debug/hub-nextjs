# 🚀 PROMPT AGENT : Transformation Hub Admin → Incubateur International Pro

## 📋 CONTEXTE ACTUEL

Je suis un agent qui vient de créer un backend admin pour gérer des projets dans Next.js. Voici l'architecture actuelle :

### Stack Technique Actuel
- **Frontend**: Next.js 15 (App Router) + React 19 + TypeScript + Tailwind CSS
- **Backend**: Next.js API Routes intégrées
- **Database**: SQLite (better-sqlite3)
- **Auth**: JWT + bcryptjs
- **Services**: ProjectService (CRUD complet)

### Structure Actuelle de la Base de Données

```sql
-- Table PROJECTS (13 projets actuellement)
CREATE TABLE projects (
  id TEXT PRIMARY KEY,
  name TEXT NOT NULL,
  slug TEXT UNIQUE NOT NULL,
  status TEXT CHECK(status IN ('live', 'coming', 'future', 'archived')),
  category TEXT NOT NULL,
  description TEXT NOT NULL,
  tech_stack TEXT NOT NULL,  -- JSON array
  port INTEGER,
  urls TEXT,                 -- JSON object
  logo TEXT,
  screenshots TEXT,          -- JSON array
  location TEXT NOT NULL,
  features TEXT,             -- JSON array
  created_at TEXT DEFAULT (datetime('now')),
  updated_at TEXT DEFAULT (datetime('now')),
  published INTEGER DEFAULT 1,
  order_index INTEGER DEFAULT 0
);

-- Table MEDIA (préparée, non utilisée)
CREATE TABLE media (
  id TEXT PRIMARY KEY,
  project_id TEXT NOT NULL,
  type TEXT CHECK(type IN ('logo', 'screenshot', 'banner')),
  filename TEXT NOT NULL,
  path TEXT NOT NULL,
  url TEXT NOT NULL,
  size INTEGER NOT NULL,
  created_at TEXT DEFAULT (datetime('now')),
  FOREIGN KEY (project_id) REFERENCES projects(id) ON DELETE CASCADE
);

-- Table USERS (admin simple)
CREATE TABLE users (
  id TEXT PRIMARY KEY,
  username TEXT UNIQUE NOT NULL,
  password_hash TEXT NOT NULL,
  role TEXT DEFAULT 'admin',
  created_at TEXT DEFAULT (datetime('now'))
);
```

### API Routes Existantes

**Auth:**
- `POST /api/auth/login` - Login JWT
- `GET /api/auth/me` - User info

**Projects:**
- `GET /api/projects` - Liste (filtres: status, published)
- `POST /api/projects` - Créer
- `GET /api/projects/:id` - Détail
- `PATCH /api/projects/:id` - Modifier
- `DELETE /api/projects/:id` - Supprimer
- `POST /api/projects/:id/publish` - Publier
- `POST /api/projects/:id/unpublish` - Dépublier
- `GET /api/projects/statistics` - Stats globales

### Dashboard Admin Actuel (React)

Fonctionnalités basiques :
- Login/Logout
- Tableau de bord avec stats (Total, Publiés, Live, Coming)
- Liste des projets avec filtres par status
- Actions : Publier/Dépublier/Supprimer
- Design moderne avec Tailwind
